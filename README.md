# 同济物理实验-latex模板
>作者2551633@tongji.edu.cn

使用方法:每次有实验, 
就复**制外面这个模板文件夹**, 
然后**figures文件夹下放实验图片**, 
data文件夹下放实验数据,
进入**latex文件夹**后,
在**实验报告.tex**文件中填写实验内容
里面的**template.....tex**文件是模板文件, 不需要修改,当然也可根据需要自定义模板文件,欢迎分享

## 不能删除的文件
- **template.....tex**
- **实验报告.tex**
- **figures文件夹**下的4个logo图片文件
